# Instagram carousel — "A week of Instagram content in 30 minutes"

A self-contained, swipeable 9-slide carousel built from the blog post. One idea per slide.

## Add your photo
Drop a square photo named **`kate.jpg`** in this folder (`/carousel/kate.jpg`).
It appears as a circular portrait on the cover and CTA slides. Until you add it,
a lime "K" placeholder shows automatically, so the page never looks broken.

The photo is the 4th image from: https://www.instagram.com/p/DWrA4dQFBxd/
(Instagram blocks automated downloads, so save it from the post and copy it in here.)

## View it
Open `/carousel/` in a browser. Navigate with the arrows, dots, arrow keys, or swipe.

## Export for Instagram
Each slide is a 1:1 square. Screenshot the `.frame` for each slide (1080×1080),
or use a full-page screenshot tool, then upload the slides as an IG carousel.

## Design
- Matches the Savage Social brand (dark canvas, lime `#c8ff00` accent)
- Readable modern type: **Inter** (Google Fonts), large sizes, generous line height
- Fully responsive; works on phone and desktop
